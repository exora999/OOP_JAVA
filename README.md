# 🏫 School Management System

A Java console application designed to manage the personnel of an educational institution. 

## 🚀 Features
* **Role Management:** Add and manage Employees, Professors, and Students.
* **Dynamic Inputs:** Interactive console interface using `Scanner`.
* **Bonus Calculation:** Specific logic applied for Professors responsible for modules.

## 🛠️ Technical Concepts Demonstrated
* **Inheritance (الوراثة):** `Employe` and `Etudiant` inherit from `Personne`, and `Professeur` inherits from `Employe`.
* **Interfaces (الواجهات):** Implementation of the `Bonus` interface for specific roles.
* **Polymorphism (تعدد الأشكال):** Overriding the `afficher()` method across different classes to display specific attributes.
* **Encapsulation:** Secure data handling using private attributes and getter/setter methods.
